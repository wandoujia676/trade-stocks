## 小金库 4.0 核心改动

### 1. 止盈止损动态计算（基于九本书理论）

**来源**：《一买即涨》《交易真相》《量学》《短线操盘》

| 因素 | 说明 |
|------|------|
| 前高位置 | 止盈目标参考20日/50日新高 |
| BOLL位置 | 参考BOLL上下轨 |
| 历史波动率 | ATR计算，反映股票真实波动 |
| 均线支撑 | MA10/MA20支撑位作为止损参考 |

**不再是统一的5%止损、15-20%止盈**，而是根据每只股票的特性动态计算。

### 2. 全市场真正筛选

- 候选池从500只扩大到800只
- 按行业分层采样（每行业最多25只）
- 移除硬编码行业龙头依赖
- 月度选股真正从全市场股票中筛选

### 3. 出击偏向高动量股票

- 技术面筛选放宽，让更多高动量股票通过
- 基本面筛选降低市值门槛（50亿→10亿）
- 动量加权：近期涨幅大的股票额外加10-20分
- 更容易选出有大幅上涨潜力的股票

### 4. 买入区间收紧

- 最佳买入区间从 ±2% 收紧到 ±1%
- 降低买入成本风险

## 系统架构 - 三层候选池

```
┌─────────────────────────────────────────────────────────────┐
│ 第一层：monthly_watchlist（100只）                           │
│ 来源：全市场5000+只 → 九本书理论初筛 → 100只                │
│ 频率：每月15号、28号更新                                    │
│ 工具：monthly_generator.py（tushare pro_basic + 批量日线）  │
├─────────────────────────────────────────────────────────────┤
│ 第二层：weekly_watchlist（20只）                             │
│ 来源：monthly的100只 → 六维度战法评分 → 20只                │
│ 频率：每周三14:00自动选，或手动随时选                        │
│ 工具：screener.py 六维度评分                                │
├─────────────────────────────────────────────────────────────┤
│ 第三层：出击（约5只）                                        │
│ 来源：weekly的20只 → 精选最佳5只                            │
│ 频率：每日出击前精选                                        │
│ 标准：信号最强（买入/加仓）、评分≥65                        │
└─────────────────────────────────────────────────────────────┘
```

### 九本书核心理论用于月度初筛

| 条件 | 评分加分 | 理论来源 |
|------|----------|----------|
| 10日内有涨停 | +20 | 《股道人生》- 涨停基因 |
| 5日涨幅>10% | +15 | 《短线操盘》- 动量 |
| 20日均线多头 | +15 | 《一买即涨》- 均线 |
| 量比>1.5 | +10 | 《量学》- 放量 |
| MACD在0轴上方 | +10 | 《交易真相》 |
| 接近20日新高 | +10 | 强势信号 |

## 核心功能

### 出击 (skill_sel)
- 综合战法6维度评分（趋势/动量/量价/形态/位置/情绪）
- 三层候选池体系：monthly → weekly → 出击
- 每日可手动出击，支持筛选买入/加仓信号
- 战法权重根据验证结果自动优化

### 验证 (skill_VER)
- 验证前一日出击的涨跌表现
- 分析成功/失败原因
- 自动更新战法权重配置
