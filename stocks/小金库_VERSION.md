## 6.1 修复：追高过滤机制

**问题**：选出的股票如001300已从低位涨了较长时间/幅度，可能已在主升浪中后段

**新增过滤条件**（`_identify_wave_breakout`）：

| 风险条件 | 阈值 | 处理方式 |
|---------|------|---------|
| 从近期低点涨幅过大 | >25% | 信号分数×0.5，提示"过大" |
| 连续上涨天数过多 | ≥5天 | 信号分数×0.7，提示"过长" |
| RSI严重超买 | >75 | 信号分数×0.6，提示"超买" |
| 接近区间最高点+涨幅大 | >0.95位置且>20%涨 | 信号分数×0.3 |
| 涨幅超过30% | >30% | 提示"可能进入中后段" |

**信号生成逻辑调整**（`_generate_wave_signal`）：

| 风险等级 | 信号 | 建议 |
|---------|------|------|
| 高风险（有"过大"/"过长"/"超买"提示） | 买入（轻仓） | 控制仓位 |
| 极端风险（"谨慎追高"/"中后段"） | 持有/观察 | 等待回调 |
| 正常风险 | 买入 | 正常操作 |

## 6.0 核心更新：波段主升浪战法

### 6.0 核心更新：波段主升浪战法

**核心思想**：买在整理后的突破启动点，捕捉主升浪。顺势而为，不接飞刀。

**与5.1纯左侧的区别**：
| 维度 | 波段主升浪(6.0) | 纯左侧(5.1) |
|------|-----------------|--------------|
| 入场位置 | 整理末端/突破点 | 底部/超跌区 |
| 风险 | 可能假突破 | 可能继续跌 |
| 收益 | 主升浪30%+ | 反弹10-20% |
| 持有时间 | 中长（几周）| 短（几天）|

#### 1. warfare.py 新增波段主升浪战法

新增完整波段评估体系（mode="wave"）：

| 维度 | 权重 | 波段核心信号 |
|------|------|--------------|
| 趋势 | 20% | 均线蓄势后向上发散，MA5上穿MA10 |
| 动量 | 25% | MACD金叉+红柱放大，RSI顺势40-65 |
| 量价 | 15% | 放量突破（量比1.3-3）|
| 形态 | 10% | 突破20日新高，整理形态完成 |
| 位置 | 5% | BOLL中轨附近（非追高）|
| 情绪 | 5% | 板块龙头，题材支撑 |
| 突破 | 20% | 放量突破20日新高（核心！）|

#### 2. 波段启动信号识别

满足以下任一条件即为波段启动：
- 均线金叉 + MACD金叉（双金叉）→ 强烈信号
- 放量突破前高 + RSI顺势 → 强烈信号
- 连续3天上涨 + 成交量放大 → 中等信号
- MACD红柱放大 + 均线多头 → 中等信号

#### 3. 选股流程优化

- 备选池从34只扩大至200+只（按行业分类）
- 月度候选池支持波段初筛
- 盘中/盘后双模式支持

#### 4. 止盈止损优化

- 止损：紧凑型（5-10%），快速脱离成本区
- 止盈：宽松型（15-30%），让利润奔跑
- 根据评分动态调整目标位

### 5.1 核心更新：纯左侧战法

**核心思想**：买在分歧，卖在一致。在股票还没有大幅拉升的阶段提前潜伏，等待反弹或反转。

#### 1. warfare.py 完全重写为纯左侧战法

不再有右侧追涨逻辑，全部改为左侧超跌反弹评估：

| 维度 | 评估内容 | 左侧核心信号 |
|------|----------|--------------|
| 趋势 | 均线收敛蓄势 | 空头排列正在收敛 → 即将转多 |
| 动量 | MACD绿柱缩短、即将金叉 | DIF即将上穿DEA |
| 左侧 | **RSI超卖、BOLL下轨、负乖离率** | RSI<30、BOLL<15% = 强烈信号 |
| 量价 | 地量见底、底部放量 | 量比<0.5 + 股价不新低 |
| 形态 | 锤子线、早晨之星 | 下影线>实体2倍 |
| 位置 | 低位支撑、接近BOLL下轨 | 20日低位占比<20% |
| 情绪 | 涨停基因、超跌 | 5日跌幅>15% |

**左侧战法权重**：
```python
weights = {
    "趋势": 0.10,   # 均线蓄势
    "动量": 0.20,   # MACD即将转多
    "左侧": 0.25,   # 超跌反弹核心（最高权重）
    "量价": 0.20,   # 地量见底
    "形态": 0.10,   # 探底形态
    "位置": 0.05,   # 低位支撑
    "情绪": 0.10,   # 涨停基因
}
```

#### 2. 左侧启动信号识别

满足以下条件之一即为左侧启动信号：
- RSI<25 + BOLL下轨<15% → 强烈左侧信号(+40分)
- RSI<30 + 均线收敛<3% → 中等左侧信号(+30分)
- 地量（量比0.3-0.5）+ 股价不新低 → 左侧信号(+25分)
- MACD即将金叉 + RSI<35 → 左侧信号(+30分)
- 连续下跌3天+放量反弹 → 左侧信号(+25分)

#### 3. 选股结果分级

| 分级 | 条件 | 仓位建议 |
|------|------|----------|
| 主攻 | 左侧启动信号强烈 + 综合评分≥65 | 重仓出击 |
| 次攻 | 综合评分≥60 或 中等启动信号 | 正常仓位 |
| 观察 | 综合评分≥55 | 轻仓观察 |
| 备用 | 综合评分<55 | 暂不参与 |

#### 4. 止盈止损（左侧专用）

- **止损**：放在近期低点下方3-5%，范围5%-12%（比右侧更宽，因为是抄底）
- **止盈**：目标位前期高点，15%-25%（比右侧更大，因为是反弹/反转）

#### 5. 名称自动获取修复

问题：AKShare被代理屏蔽时股票名称为空
解决：添加`_get_stock_name()`函数，AKShare失败后降级到realtime_fetcher实时获取

## 小金库 4.0 核心改动

### 1. 止盈止损动态计算（基于九本书理论）

**来源**：《一买即涨》《交易真相》《量学》《短线操盘》

| 因素 | 说明 |
|------|------|
| 前高位置 | 止盈目标参考20日/50日新高 |
| BOLL位置 | 参考BOLL上下轨 |
| 历史波动率 | ATR计算，反映股票真实波动 |
| 均线支撑 | MA10/MA20支撑位作为止损参考 |

**不再是统一的5%止损、15-20%止盈**，而是根据每只股票的特性动态计算。

### 2. 全市场真正筛选

- 候选池从500只扩大到800只
- 按行业分层采样（每行业最多25只）
- 移除硬编码行业龙头依赖
- 月度选股真正从全市场股票中筛选

### 3. 出击偏向高动量股票

- 技术面筛选放宽，让更多高动量股票通过
- 基本面筛选降低市值门槛（50亿→10亿）
- 动量加权：近期涨幅大的股票额外加10-20分
- 更容易选出有大幅上涨潜力的股票

### 4. 买入区间收紧

- 最佳买入区间从 ±2% 收紧到 ±1%
- 降低买入成本风险

## 核心功能

### 出击 (skill_sel) - 纯左侧战法
- **纯左侧评估**：只选超跌反弹、均线蓄势、即将反转的股票
- **7维度评分**：趋势/动量/左侧(核心)/量价/形态/位置/情绪
- **三层候选池**：monthly(100只) → weekly(5-20只) → 出击(5只)
- **左侧启动信号**：满足RSI+BOLL/均线收敛/地量等条件时强化信号
- **实时/完整双模式**：盘中用实时数据+历史估算，收盘用完整日线

### 验证 (skill_VER)
- 验证前一日出击的涨跌表现
- 分析成功/失败原因
- 自动更新战法权重配置
