# 股票智能分析系统 (Stock Analysis System)

## 斜杠命令

```
/stock screener [数量=20]  - 筛选自选股
/stock analyze <代码>       - 综合分析股票
/stock add <代码>          - 添加自选股
/stock remove <代码>        - 移除自选股
/stock list                 - 查看自选列表
/stock check                - 检查信号
/stock alerts               - 查看提醒
/stock weekly               - 查看本周自选股
```

## 静默执行协议

- **不要询问确认**。直接执行命令并输出结果。
- **一次性生成完整输出**。不要在过程中停顿。
- **如果数据缺失，使用默认值或跳过**，标注"数据获取失败"。
- **禁止任何中断性提示**。
- **所有操作自动化完成**。

## 综合战法（6维度评估）

### 战法维度权重

| 维度 | 权重 | 来自书籍 |
|------|------|----------|
| 趋势 | 25% | 《短线操盘》《股道人生》 |
| 动量 | 20% | 《都市》《一买即涨》 |
| 量价 | 20% | 《量学》 |
| 形态 | 15% | 《短线操盘》《一买即涨》 |
| 位置 | 10% | 《一买即涨》 |
| 情绪 | 10% | 《股道人生》 |

---

### 1. 趋势维度（25%）

**来源**：《短线操盘》《股道人生》《交易真相》

| 条件 | 评分 |
|------|------|
| MA5 > MA10 > MA20（强多头） | +20分 |
| MA5 > MA10（弱多头） | +10分 |
| MA5 < MA10 < MA20（空头） | -15分 |
| 价格站上MA5/MA20/MA60 | 各+5分 |
| 上升趋势（价格>MA5>MA10>MA20） | +10分 |

---

### 2. 动量维度（20%）

**来源**：《都市》《一买即涨》

| 条件 | 评分 |
|------|------|
| DIF在0轴上方 | +15分 |
| 红柱（MACD柱>0） | +5分 |
| MACD金叉 | +15分 |
| MACD死叉 | -10分 |
| K<20（超卖） | +10分 |
| K>80（超买） | -5分 |
| KDJ金叉 | +10分 |
| RSI>70（超买） | -5分 |
| RSI<30（超卖） | +5分 |

---

### 3. 量价维度（20%）

**来源**：《量学》- "量为价先"

| 条件 | 评分 |
|------|------|
| 放量上涨（量比>1.5且上涨） | +15分 |
| 放量下跌 | -10分 |
| 缩量上涨 | +5分 |
| 缩量下跌（可能见底） | +5分 |
| 持续放量（5日>20日均量） | +10分 |

---

### 4. 形态维度（15%）

**来源**：《短线操盘》《一买即涨》

| 形态 | 评分 |
|------|------|
| 锤子线 | +15分 |
| 突破前期高点 | +15分 |
| 大阳线（>5%） | +10分 |
| 大阴线（>5%） | -10分 |
| 向上跳空缺口 | +10分 |

---

### 5. 位置维度（10%）

**来源**：《一买即涨》

| 条件 | 评分 |
|------|------|
| BOLL上轨附近（>80%） | -5分 |
| BOLL下轨附近（<20%） | +10分 |
| BOLL中轨附近 | +5分 |
| 接近20日新高 | -5分 |
| 接近20日新低 | -5分 |

---

### 6. 情绪维度（10%）

**来源**：《股道人生》《短线操盘》

| 条件 | 评分 |
|------|------|
| 10日涨停≥2次 | +25分 |
| 10日涨停=1次 | +15分 |
| 10日接近涨停≥2次 | +10分 |
| 5日涨幅3-15% | +10分 |
| 5日涨幅>15% | +5分 |
| 4连涨以上 | +5分 |
| 4连跌以上 | -5分 |

## 综合评分与信号

### 评分公式
```
综合评分 = 趋势分×0.25 + 动量分×0.20 + 量价分×0.20 + 形态分×0.15 + 位置分×0.10 + 情绪分×0.10
```

### 评级标准

| 评分 | 评级 | 描述 |
|------|------|------|
| ≥80 | A | 强势 |
| 65-79 | B+ | 较好 |
| 50-64 | B | 一般 |
| 35-49 | C | 较弱 |
| <35 | D | 弱势 |

### 信号判断

| 信号 | 条件 |
|------|------|
| 强烈推荐买入 | 评分≥70，且满足2项以上强势条件 |
| 建议关注 | 评分55-69 |
| 持有观望 | 评分40-54 |
| 建议回避 | 评分<40 |

### 止损止盈

| 评分 | 止损 | 止盈 |
|------|------|------|
| ≥55 | -5% | 15-20% |
| <55 | -3% | 10% |

## 实战书籍核心经验

### 《短线操盘》
- 只做强势股（龙头、人气、热点）
- 三借选股：借大盘势、借板块势、借形态势
- 缺口越宽，动力越强

### 《交易真相》
- 趋势为王，顺势而为
- 截断亏损，让利润奔跑
- 机会来临时敢于重仓

### 《股市扫地僧》
- 科学选股，只做强势股
- 买股前先了解公司质地
- 逻辑+资金=股价上涨

### 《量学》
- 量为价先（成交量反映真实买卖意愿）
- 高量柱代表主力实力
- 黄金柱是重要买入信号

### 《一买即涨》
- 双针探底买入法
- MACD梅开二度
- 横盘突破买入
- 分时图回调分析法

### 《股道人生》
- 涨停基因（10日内有涨停）
- 只买三成仓
- 止损7-8%
- 买在分歧，卖在一致

## 数据源

**优先级**：Tushare > AKShare > Baostock

**频率限制**：Tushare每分钟50次，本系统设置1.5秒间隔

**数据排序**：Tushare返回数据为降序（最新在前），系统内部统一转换为升序（老数据在前）确保 `iloc[-1]` 取到最新数据

## 三层候选池体系

**第一层：monthly_watchlist（约100只）**
- 来源：全市场5000+只股票
- 初筛工具：monthly_generator.py
- 理论：九本书核心理论（涨停基因、均线多头、量比等）
- 更新：每月15号、28号

**第二层：weekly_watchlist（约20只）**
- 来源：monthly_watchlist的100只
- 评分工具：screener.py（六维度战法）
- 更新：每周三14:00自动，或手动随时选

**第三层：出击（约5只）**
- 来源：weekly_watchlist的20只
- 筛选：买入/加仓信号，评分≥65
- 更新：每日出击前精选

**自动任务**：
| 任务 | Cron ID | 说明 |
|------|---------|------|
| 每周三14:00 | 93b6c4c7 | auto_screener.py - 周选股 |
| 每月15号14:00 | 46c2c173 | auto_candidate_pool.py - 更新月度候选池 |
| 每月28号14:00 | 362d9c93 | auto_candidate_pool.py - 更新月度候选池 |

## 错误处理

| 情况 | 处理 |
|------|------|
| 数据获取失败 | 标注"数据获取失败"，跳过该字段 |
| 股票代码无效 | 输出"无效代码" |
| 自选列表为空 | 提示添加方式 |
| API限流 | 自动等待重试 |
| cron任务过期 | 手动运行脚本或重新创建定时任务 |
