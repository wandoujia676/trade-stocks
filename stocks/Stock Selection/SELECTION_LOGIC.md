# 选股逻辑说明文档

## 一、概述

本系统基于9本实战炒股书籍的核心经验，开发了一套**多维度综合战法选股系统**。

### 数据来源
- **Tushare Pro**（主数据源）
- **AKShare**（备选）
- **Baostock**（最后备选）

### 核心维度
1. **趋势维度** - 均线系统、趋势判断
2. **动量维度** - MACD、KDJ、RSI
3. **量价维度** - 成交量、量比
4. **形态维度** - K线形态、突破形态
5. **位置维度** - BOLL、相对位置
6. **情绪维度** - 涨停基因、短期强度

---

## 二、各维度评估逻辑

### 1. 趋势维度（权重25%）

**来源**：《短线操盘》《股道人生》《交易真相》

#### 均线系统评估
| 条件 | 评分变化 |
|------|----------|
| MA5 > MA10 > MA20（强多头） | +20分 |
| MA5 > MA10（弱多头） | +10分 |
| MA5 < MA10 < MA20（空头） | -15分 |
| 价格站上MA5 | +5分 |
| 价格站上MA20 | +5分 |
| 价格站上MA60 | +5分 |

#### 趋势判断
- **上升趋势**：MA5 > MA10 > MA20，价格在各均线上方 → +10分
- **下降趋势**：空头排列 → -10分
- **震荡整理**：其他情况

### 3. 量价维度（权重20%）

**来源**：《量学》核心思想 - "量为价先"

#### 量价信号
| 条件 | 评分变化 |
|------|----------|
| 放量上涨（量比>1.5且上涨） | +15分 |
| 放量下跌 | -10分 |
| 缩量上涨 | +5分 |
| 缩量下跌（可能见底） | +5分 |

#### 持续放量
- 连续5日成交量大于20日均量 → +10分

#### 活跃度
- 5日均成交额较大 → +5分
- 成交低迷 → -5分

### 5. 位置维度（权重10%）

**来源**：《一买即涨》《短线操盘》

#### BOLL评估
| 条件 | 评分变化 |
|------|----------|
| BOLL上轨附近（>80%） | -5分（注意回调） |
| BOLL下轨附近（<20%） | +10分（可能反弹） |
| BOLL中轨附近 | +5分 |

#### 相对位置
- 接近20日新高 → -5分（可能有压力）
- 接近20日新低 → -5分（可能继续下跌）

---

### 6. 情绪维度（权重10%）

**来源**：《股道人生》《短线操盘》

#### 涨停基因（最重要）
| 条件 | 评分变化 |
|------|----------|
| 10日内涨停≥2次 | +25分 |
| 10日内涨停=1次 | +15分 |
| 10日内接近涨停≥2次 | +10分 |

#### 短期强度
| 条件 | 评分变化 |
|------|----------|
| 5日涨幅3-15% | +10分（健康上涨） |
| 5日涨幅>15% | +5分（强势但注意回调） |
| 5日跌幅>10% | -10分 |

#### 连涨/连跌
- 4连涨以上 → +5分
- 4连跌以上 → -5分

## 三、综合评分计算

### 评分公式
```
综合评分 = 趋势分×0.25 + 动量分×0.20 + 量价分×0.20 + 形态分×0.15 + 位置分×0.10 + 情绪分×0.10
```

### 评级标准
| 综合评分 | 评级 | 描述 |
|---------|------|------|
| ≥80 | A | 强势 |
| 65-79 | B+ | 较好 |
| 50-64 | B | 一般 |
| 35-49 | C | 较弱 |
| <35 | D | 弱势 |

## 四、交易信号

### 信号判断规则

#### 强烈推荐买入（评分≥70）
条件：
- 综合评分≥70
- 同时满足以下至少2项：
  - 均线多头排列
  - MACD金叉
  - 放量上涨
  - 突破新高
  - 有涨停基因

#### 建议关注（评分55-69）
条件：
- 综合评分55-69
- 部分维度表现良好

#### 持有观望（评分40-54）
条件：
- 综合评分40-54
- 无明显方向

#### 建议回避（评分<40）
条件：
- 综合评分<40
- 均线空头排列
- 无涨停基因

## 六、实战经验总结（来自书籍）

### 《短线操盘》- 核心要点
1. 只做强势股（龙头、人气、热点）
2. 三借选股：借大盘势、借板块势、借形态势
3. 缺口越宽，动力越强
4. 涨停位置分析：MA100之下的涨停更有意义

### 《交易真相》- 核心要点
1. 趋势为王，顺势而为
2. 截断亏损，让利润奔跑
3. 不要追求买在最低卖在最高
4. 机会来临时敢于重仓

### 《股市扫地僧》- 核心要点
1. 科学选股，只做强势股
2. 买股前先了解公司质地
3. 逻辑+资金=股价上涨
4. 坚持做正确的事

### 《量学》- 核心要点
1. 量为价先（成交量反映真实买卖意愿）
2. 高量柱代表主力实力
3. 黄金柱是重要买入信号
4. 缩量下跌可能是假跌

### 《一买即涨》- 核心要点
1. 双针探底买入法
2. MACD梅开二度
3. 横盘突破买入
4. 分时图回调分析法

### 《股道人生》- 核心要点
1. 涨停基因（10日内有涨停）
2. 只买三成仓
3. 止损7-8%
4. 买在分歧，卖在一致
