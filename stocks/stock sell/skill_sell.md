# 卖出决策系统 (Sell Decision System)

## 斜杠命令

```
/sell analyze <股票@买入价>      - 分析单只持仓卖出时机
/sell analyze <股票@买入价> <股票@买入价> - 分析多只持仓
```

- **不要询问确认**。直接执行命令并输出结果。
- **一次性生成完整输出**。不要在过程中停顿。
- **如果数据缺失，使用默认值或跳过**，标注"数据获取失败"。
- **禁止任何中断性提示**。
- **所有操作自动化完成**。

## 卖出信号检测（8种见顶信号）

来自九本书核心理论：

| 信号类型 | 来源书籍 | 权重 | 说明 |
|----------|----------|------|------|
| 高位十字星 | 《一买即涨》 | 20% | 上下影线较长，实体很小，出现在相对高位 |
| 高位长上影线 | 《一买即涨》 | 15% | 上影线>5%，冲高回落 |
| 高位大阴线 | 《一买即涨》 | 15% | 实体>5%，高开低走 |
| 均线死叉 | 《一买即涨》 | 15% | MA5下穿MA10 |
| MACD死叉 | 《一买即涨》《都市》 | 15% | DIF下穿DEA |
| 缩量滞涨 | 《量学》 | 10% | 量能萎缩但价格不创新高 |
| 跌破20日均线 | 《短线操盘》 | 10% | 有效跌破MA20 |
| 量价背离 | 《量学》 | 10% | 价格创新高但量能萎缩 |

---

## 卖出评分公式

```
综合卖出评分 = 技术见顶信号分×0.30 + 持仓收益分×0.25 + 趋势转折分×0.20 + 量价背离分×0.15 + 持仓周期分×0.10
```

### 评分标准

| 评分 | 建议 | 说明 |
|------|------|------|
| >=80 | **强烈卖出** | 多个见顶信号，建议清仓 |
| 65-79 | **考虑减仓** | 出现危险信号，建议分批卖出 |
| 50-64 | **谨慎持有** | 仍有盈利但需关注变化 |
| <50 | **继续持有** | 未见明显卖出信号 |

## 分档卖出方案

### 规则（来自九本书）

**持仓盈利 > 20%：**
- 第一档 50%：回调至止盈位附近，锁利
- 第二档 30%：再次上涨5%后卖出
- 第三档 20%：博取更大趋势（不破止损不出）

**持仓盈利 10-20%：**
- 第一档 60%：触及止盈位卖出
- 第二档 40%：留40%观察趋势

**持仓盈利 5-10%：**
- 第一档 30%：回调至成本附近减仓
- 第二档 70%：持有等待更高目标

**持仓亏损：**
- 触及止损位坚决止损（《交易真相》截断亏损）
- 未达止损可观望

### 《股道人生》仓位原则
- 只买三成仓 → 卖出时也建议分批
- 买在分歧卖在一致 → 在市场亢奋时分批离场

---

## 止盈止损基准

| 综合评分 | 止损 | 止盈 |
|----------|------|------|
| >=55分 | -5% | 15-20% |
| <55分 | -3% | 10% |

**《股道人生》额外规则**：止损统一设置 7-8%

---

## 持仓周期建议

| 持仓天数 | 策略建议 |
|----------|----------|
| 1-3天 | 短线策略：严格止损3-5%，涨5%+考虑卖出 |
| 4-10天 | 偏短线：看是否放量突破，量能萎缩考虑卖 |
| 11-30天 | 中线策略：趋势未破可持有，跌破20日线卖出 |
| >30天 | 长线思路：基本面向好可持有，趋势破位再卖 |

## 实战书籍卖出理论

### 《一买即涨》
**短线卖出方法：**
- **高位十字星**：出现在相对高位，伴随巨大成交量，十字星是主力出货标志，要坚决卖出
- **高位长上影**：冲高回落伴随巨大成交量，上影线越长见顶信号越明确
- **高位长下影**：迷惑性大，主力诱多出货行为，要坚决卖出
- **高位大阴线**：高开低走，实体大于5%，配合成交量放大，是主力出货行为

**分时图卖出技法：**
- 均价线拐头向下卖出
- 不破前高卖出（后一波不能超过前一波高点）
- 跌破前低卖出
- 跌破均价线卖出
- 反弹至均价线卖出

**分钟图卖出技法：**
- 5线死叉10线即为卖出信号
- 配合MACD死叉、下跌带量准确率更高
- 多周期空头共振是暴跌信号

### 《交易真相》
- 趋势为王，顺势而为
- **截断亏损，让利润奔跑** - 止损必须要设置
- 最好不被动等待市场去测试止损
- 进场后如果经常要容许价格走出一段距离才掉头测试止损，说明系统设计有问题
- 理想的止损是防止特殊情况的

### 《股道人生》
- 买在分歧，卖在一致
- 只买三成仓
- **止损7-8%**
- 10日有涨停是强势股

### 《量学》
- 量为价先（成交量反映真实买卖意愿）
- **高量柱后缩量滞涨是见顶信号**
- 卖在买先、价在量先、庄在散先
- 三先规律在卖出中的运用

### 《短线操盘》
- 早晨之星卖出
- 跌破重要均线卖出
- 只做强势股（卖出时也选择强势股出货时机）
